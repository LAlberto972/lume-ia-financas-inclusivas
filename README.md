# lume-ia-financas-inclusivas
Projeto desenvolvido para o Desafio DIO + Microsoft Copilot. O Lume IA é um app de finanças pessoais acessível, multilíngue e com IA, criado com apoio do Copilot e gerado no Lovable. O repositório contém o README completo, PRD, prompt final, prints das telas e reflexão sobre o processo.
